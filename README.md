# Kasumi

**プライバシー重視のmacOS画像・PDF編集アプリ**

すべての処理はローカルで完結。ネットワーク通信は一切行いません。

---

## 特徴

- **完全ローカル処理** — 画像・PDFデータがインターネットに送信されることはありません
- **モザイク処理** — 範囲選択またはブラシでモザイク・ぼかし・フロストガラス・カラー塗りつぶしを適用
- **背景透過** — クリックで選択した色に近い連続領域を透明化。色範囲をスライダーでリアルタイム調整
- **切り抜き（トリミング）** — 矩形選択で画像を切り抜き
- **PDF対応** — 複数ページのPDFを読み込み、ページごとにモザイク加工・保存が可能
- **リアルタイムプレビュー** — モザイク・背景透過ともに、確定前にプレビューで確認・調整可能
- **Undo / Redo** — 無制限の編集履歴（画像・PDFページごとに独立管理）
- **グローバルショートカット** — `Cmd+Shift+K` でどこからでもアプリを起動
- **カスタマイズ可能なショートカット** — 設定画面からエディタのショートカットキーを自由に変更

## 動作環境

- macOS 13 Ventura 以上
- Apple Silicon / Intel Mac

## インストール

### GitHub Releases からダウンロード

1. [Releases](../../releases) ページから最新の `Kasumi.dmg` をダウンロード
2. DMGファイルを開き、`Kasumi.app` を「アプリケーション」フォルダにドラッグ
3. **初回起動前に**ターミナルで以下を実行（Apple Developer ID 未署名のため macOS Gatekeeper にブロックされます）:
   ```bash
   xattr -cr /Applications/Kasumi.app
   ```
4. 初回起動時は右クリック → 「開く」を選択

### ソースからビルド

```bash
git clone https://github.com/ShiningWank0/Kasumi.git
cd Kasumi
open Kasumi.xcodeproj
```

Xcodeで `Cmd+R` を押してビルド・実行します。

## 使い方

### 画像を開く

- アプリのメイン画面から「Open File」ボタンで画像/PDFを選択
- クリップボードに画像がある状態で「From Clipboard」ボタンをクリック
- グローバルショートカット `Cmd+Shift+K` でFinderで選択中のファイルまたはクリップボード画像を直接開く

### ツール

| ツール | 説明 | デフォルトキー |
|--------|------|---------------|
| 切り抜き | 矩形選択で画像をトリミング | `C` |
| モザイク | 範囲選択またはブラシでモザイク処理 | `M` |
| 背景透過 | クリックした色の連続領域を透明化 | `T` |
| 元に戻す | 直前の操作を取り消し | `Cmd+Z` |
| やり直し | 取り消した操作を再実行 | `Cmd+Shift+Z` |
| 保存 | 名前を付けて保存 | `Cmd+S` |
| 上書き保存 | 元ファイルに上書き | `Cmd+Shift+S` |

### モザイクの使い方

1. 「モザイク」ボタンをクリック
2. 「範囲」または「ブラシ」を選択
3. 画像上で範囲をドラッグまたはブラシで描画
4. プレビューが表示される（赤い破線で外枠が点滅）
5. ドロップダウンでエフェクト種類、スライダーで粗さ・太さを調整（リアルタイム反映）
6. `Enter` で確定、`Esc` または `Cmd+Z` で取り消し

### 背景透過の使い方

1. 「背景透過」ボタンをクリック
2. 透明にしたい領域をクリック
3. プレビューが表示される（赤いフチが点滅）
4. 「色範囲」スライダーで透過する色の範囲を調整（リアルタイム反映）
5. `Enter` で確定、`Esc` または `Cmd+Z` で取り消し

### PDF編集

- PDFを開くとページナビゲーション（`< 1/5 >`）が表示される
- ページごとにモザイク処理が可能
- 切り抜き・背景透過はPDFでは無効
- 保存時、モザイクをかけた領域の埋め込みテキストは削除される（復元不可）

## 対応ファイル形式

| 形式 | 読み込み | 書き出し |
|------|---------|---------|
| PNG | O | O |
| JPEG | O | O |
| TIFF | O | O |
| HEIC | O | - |
| PDF | O（複数ページ） | O |

## 設定

メニューバーの「Kasumi > Settings...」から設定画面を開けます。

- **General** — デフォルト書き出し形式、JPEG品質、Dock表示の切り替え
- **Shortcuts** — グローバルショートカット、エディタショートカットのカスタマイズ
- **About** — バージョン情報

## プライバシー

- ネットワーク通信: **一切なし**
- ファイルアクセス: ユーザーが明示的に開いた/保存したファイルのみ
- クリップボード: ショートカット起動時のみ読み取り
- テレメトリ/クラッシュレポート: **なし**
